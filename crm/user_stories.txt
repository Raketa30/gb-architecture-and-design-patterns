User stories для CRM строительной фирмы

OFFICE MANAGER
    -видит планируемые/активные/завершенные объекты
    -видит список клиентов и относящиеся к ним объекты
    -видит список рабочих


PROJECT MANAGER
    -видит свои активные объекты
    -видит персонал задействованный на объекте
    -видит список свободных рабочих


WORKER
    -видит запланированные работы по активному объекту
    -видит список завершенных работ и выплат по ним
    -видит страницу заявки на расходный материал

CUSTOMER
    -видит список своих объектов
    -видит список активных/выполненных/планируемых работ на объекте
    -видит календарь планируемых работ
